M.create_video_mode (width, height, bpp)
Create a video_mode object to be used by M.create_window.

Parameters

  • width: Width of the window (in pixels, default: 800).
  • height: Height of the window (in pixels, default: 600).
  • bpp: Bits per pixel (default: 32).
M.create_video_settings (depth, stencil, antialiasing)
Create a video_settings object to be used by M.create_window.

Parameters

  • depth: Bits of the depth buffer (default: 24).
  • stencil: Bits of the stencil buffer (default: 8).
  • antialiasing: Level of the antialiasing (default: 0).
M.create_window (video_mode, title, style, video_settings)
Create a window for graphical applications. Use M.create_video_mode and M.create_video_settings to build the arguments for this function.

Parameters

  • video_mode: Video mode object.
  • title: String object, title of the window (if not fullscreen).
  • style: Style of the window (resizeable, closeable, fullscreen) TO BE IMPLEMENTED.
  • video_settings: Video settings object.

Return value:

Window object.
