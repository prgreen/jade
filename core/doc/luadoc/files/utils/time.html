Functions

M.begin_alloted_time (t)
Execute a loop for a minimum amount of time. Give alloted time then use M.is_alloted inside a while loop so that the loop will be executed for a given amount of time.

Parameters

  • t: Time (in seconds)
M.begin_wait (t)
Wait for a certain amount of time. Use this function in conjunction with M.end_wait in order to execute a block of instructions in at least a certain amount of time. Can be used to get a fixed framerate. Do not nest.

Parameters

  • t: Time to wait (in seconds).
M.end_wait ()
Wait for a certain amount of time. Use with M.begin_wait.
M.is_alloted ()
Conditional used in a while loop. Useful to loop for a certain amount of time. Used in conjunction with M.begin_alloted_time.

Return value:

false as soon as time is up, true otherwise
M.sleep (t)
Sleep for a certain amount of time, in seconds. Beware, this also blocks everything else in the program, so it shouldn't be called from within a coroutine...

Parameters

  • t: Time to sleep, in seconds (can be float)
M.time ()
Get current timestamp. Useful to get absolute time references and differences. Accurate to the 100 ns.

Return value:

Timestamp in seconds (seconds elapsed since 1970 and counting...)
